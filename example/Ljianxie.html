质点M的实时运动分解
运动过程分析：
0 ~ 2.0s：两波均未到达M点，M静止，路程 = 0。
2.0 ~ 4.0s：右波先到达M（传播2m，耗时2s）。M以振幅 6cm 振动，半个周期路程 = 12cm。
t = 4.0s：左波到达M（传播4m，耗时4s）。右波使M回到平衡位置速度向上，左波使M速度向下，两波反相叠加，干涉减弱。
4.0 ~ 8.0s：合振幅 = |10−6| = 4cm，一个周期路程 = 16cm。
0~8s 总路程 = 28cm。
